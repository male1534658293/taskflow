# Changelog

## v2.1.5 (2026-03-26)

### 修复
- 设置页检测到新版本后，支持直接下载更新与安装重启
- 移除“请在弹窗中选择是否更新”“请回到右下角更新浮窗执行安装”这类断裂提示，改为就地完成更新流程

## v2.1.4 (2026-03-26)

### 修复
- Google Calendar OAuth scope 补齐，新增授权不足错误归一化提示
- 新建任务解析预览支持仅输入时间时显示“今天 + 时间”
- 悬浮窗点击穿透支持再次点击关闭，保留透明度设置入口
- 悬浮窗任务列表直接展示标签内容，不再只显示标签数量

### 优化
- 主创建框与悬浮窗新增已有标签联想，输入 `#` / `@` 可快速补全
- 今天页与悬浮窗都支持把今天未完成任务迁移到明天
- 移除菜单栏托盘入口，侧边栏改为仅保留浮窗快捷键提示

## v1.0.0 (2026-03-18)

### 新功能
- 今日视图：焦点模式、任务排序、明日预览
- 收件箱、看板、甘特图、统计、成就多视图
- 日历视图：中国法定节假日 + 调休 + 周末标注
- 浮窗模式：4 档尺寸、番茄钟、迷你日历、右键菜单、虚拟滚动
- 重复任务：每天 / 工作日 / 每周 / 每月 / 每年
- Google Calendar 双向同步（OAuth 2.0 Desktop 流程）
- 工作日系统通知提醒（自动跳过节假日）
- NLP 自然语言输入（日期、优先级、标签、重复规则）
- 子任务支持：子任务全部完成时自动完成父任务
- 批量操作：多选 + 批量完成 / 删除 / 修改优先级
- 自动更新（基于 GitHub Releases）
