# Changelog

## v1.0.0 (2026-03-18)

### 新功能
- 今日视图：焦点模式、任务排序、明日预览
- 收件箱、看板、甘特图、统计、成就多视图
- 日历视图：中国法定节假日 + 调休 + 周末标注
- 浮窗模式：4 档尺寸、番茄钟、迷你日历、右键菜单、虚拟滚动
- 重复任务：每天 / 工作日 / 每周 / 每月 / 每年
- Google Calendar 双向同步（OAuth 2.0 Desktop 流程）
- 工作日系统通知提醒（自动跳过节假日）
- NLP 自然语言输入（日期、优先级、标签、重复规则）
- 子任务支持：子任务全部完成时自动完成父任务
- 批量操作：多选 + 批量完成 / 删除 / 修改优先级
- 自动更新（基于 GitHub Releases）
